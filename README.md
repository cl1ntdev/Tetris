# Tetris

A simple implementation of the classic Tetris game in a C# console application.

## How to Play

-   **Left Arrow**: Move the block left.
-   **Right Arrow**: Move the block right.
-   **R**: Restart the game.
-   **Q**: Quit the game.

## Setup and Run

This project is built with .NET 8.

### Prerequisites

-   [.NET 8.0 SDK](https://dotnet.microsoft.com/download/dotnet/8.0)

### Running the game

1.  Clone the repository:
    ```bash
    git clone <repository-url>
    ```
2.  Navigate to the project directory:
    ```bash
    cd Tetris
    ```
3.  Run the application:
    ```bash
    dotnet run
    ```
